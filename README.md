# Billy

> Every gentleman plays billiards, but someone who plays billiards too well, is no gentleman.

---

## Development

### Frontend

Node version:
`v18.*`

#### Running the project locally:

```
cd billy/frontend/
npm install
npm run dev
```

### Backend

Dotnet version: `7.0.*`

```
cd billy/backend/Billy-BE
dotnet run
```
